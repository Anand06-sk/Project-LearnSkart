ET3491 Important Topics (Unit-wise)

These are the most important unit-wise topics for ET3491 (Embedded Systems and IoT Design - R2021). Practice these topics regularly to strengthen core concepts and improve exam performance.

Unit I – Core Embedded Systems Architecture

  • Compare CISC vs RISC architectures with advantages and applications
  • Explain 8051 microcontroller architecture and instruction execution process
  • Describe timers and serial communication in 8051
  • Explain assembler directives with examples
  • Discuss ARM instruction set features and classifications
  • Explain ARM processor operating modes and their significance

Unit II – Operating Systems and RTOS

  • Define multitasking and explain its importance in embedded systems
  • Explain deadlock and conditions for its occurrence
  • Solve CPU scheduling (FCFS & Round Robin) with Gantt chart and analysis
  • Explain task states in RTOS with state transition diagram
  • Analyze conditions for task switching (running → ready / waiting)

Unit III – IoT Architecture and Concepts

  • Explain IoT architecture and challenges/issues
  • Compare IoT vs M2M communication
  • Discuss IoT communication models (Request-Response, Publish-Subscribe)
  • Explain IoT levels and real-world applications (e.g., agriculture)

Unit IV – Raspberry Pi and IoT Applications

  • Explain GPIO pins of Raspberry Pi and their usage
  • Design basic IoT systems using Raspberry Pi
  • Develop Python-based automation programs for embedded applications
  • Explain event-driven IoT systems (e.g., switch-triggered actions)

Unit V – Practical Implementation and Integration

  • Integrate embedded systems with IoT platforms
  • Explain real-time data processing in IoT applications
  • Discuss security considerations in embedded and IoT systems
  • Analyze power management techniques in IoT devices
  • Explain wireless communication protocols for IoT (WiFi, Bluetooth, Zigbee)
