EC3492 Important Topics (Unit-wise)

These are the most important unit-wise topics for EC3492 (Digital Signal Processing - R2021). Practice these topics regularly to strengthen core concepts and improve exam performance.

Unit I – Discrete Fourier Transform & Fast Fourier Transform

  • Decimation In Time Fast Fourier Transform algorithm (Radix-2 and Radix-4 butterfly)
  • Circular convolution using Discrete Fourier Transform and Inverse Discrete Fourier Transform
  • Discrete Fourier Transform computation using Fast Fourier Transform
  • Magnitude and phase response of discrete systems

Unit II – Infinite Impulse Response Filter Design

  • Finite Impulse Response vs Infinite Impulse Response filters (comparison)
  • Butterworth and Chebyshev filter characteristics
  • Digital filter design using bilinear transformation

Unit III – Finite Impulse Response Filter Design

  • Finite Impulse Response filter design using window methods
  • Window functions (Hamming, Hanning, etc.)
  • Design of filters using window technique (numericals)
  • Finite word length effects in Finite Impulse Response filters

Unit IV – Finite Word Length Effects

  • Quantization noise (input and output effects)
  • Finite word length effects in digital filters
  • Effect of quantization on pole locations
  • Noise analysis in recursive systems

Unit V – Multirate Signal Processing & Applications

  • Sampling rate conversion (interpolation and decimation)
  • Polyphase structures (interpolator and decimator)
  • Digital Signal Processor architecture
  • Adaptive filters (concept + applications)
