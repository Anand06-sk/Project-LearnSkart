About EC3351 Control Systems

EC3351 is a core subject for Anna University Semester 3 students, focusing on the fundamentals of control systems. The subject covers system modeling, time and frequency response, stability analysis, and state space methods. These EC3351 notes are designed to help you understand control concepts, analyze system behavior, and prepare for advanced engineering courses and exams.

EC3351 Important Topics (Unit-wise)

These are the most important unit-wise topics for EC3351 (Control Systems - R2021). Practice these topics regularly to strengthen core concepts and improve exam performance.

Unit I – System Components & Representation

  • Transfer function models (electrical and mechanical systems)
  • Block diagram reduction techniques
  • Signal flow graphs and Mason’s gain formula
  • Feedback control system concepts
  • DC servo system basics

Unit II – Time Response Analysis

  • First order and second order system response
  • Time domain specifications (rise time, settling time, overshoot)
  • Steady state error and error constants
  • Effect of poles and zeros
  • PID controller (PI, PD, PID design concepts)

Unit III – Frequency Response Analysis

  • Bode plot (construction and interpretation)
  • Polar plot basics
  • Nyquist plot and stability
  • Frequency response of second order system
  • Design of compensators (lead, lag, lag-lead)

Unit IV – Stability Analysis

  • Routh-Hurwitz stability criterion
  • Root locus (construction and analysis)
  • Nyquist stability criterion
  • Concept of stability

Unit V – State Space Analysis

  • State space representation
  • Conversion between transfer function and state model
  • Controllability and observability
  • Solution of state equations
  • State feedback control basics
Practice these questions regularly and revise unit by unit to build confidence for university exams.
