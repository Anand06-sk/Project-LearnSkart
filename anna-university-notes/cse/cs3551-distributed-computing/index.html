CS3551 Important Topics (Unit-wise)

Distributed Computing is essential for understanding how modern applications are built to be scalable, reliable, and available across multiple machines. The topics below are arranged unit wise to help students revise core ideas, practice exam topics, and prepare with confidence.

Unit I – Distributed Models & Computation

  • Passing S/M vs Shared Memory S/M
  • Model of distributed computations, global state
  • Designing issues, challenges and examples of DS

Unit II – Event Ordering & Consistency

  • CO (Causal Order) - full topic
  • Message ordering paradigms
  • Snapshot algorithms for FIFO channels

Unit III – Synchronization Algorithms

  • Ricart-Agrawala algorithm, Maekawa's algorithm
  • Lamport's algorithm
  • Knapp's classification (may be Part C)

Unit IV – Fault Tolerance & Recovery

  • Issues in failure recovery, Checkpoint-based recovery
  • Algorithm for asynchronous checkpointing and recovery
  • Log-based rollback recovery

Unit V – Cloud Computing & Services

  • Characteristics of CC deployment models
  • Virtualization, monitoring and cloud service platform
  • Remote storage, app services
Practice regularly, revise each unit step by step, and focus on concepts as well as problem-solving for better exam results.
