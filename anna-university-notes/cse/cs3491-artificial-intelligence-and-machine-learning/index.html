CS3491 - Artificial Intelligence and Machine Learning Important Questions (Unit-wise)

Artificial Intelligence and Machine Learning helps students build strong problem-solving skills for real-world applications. A unit-wise study method improves understanding of search algorithms, Bayesian models, regression, clustering, and neural networks for both theory and algorithm-based exam questions.

Unit I - Problem Solving

This unit covers the core problem-solving methods in AI. Focus on search strategies and CSP-based reasoning, as these are frequent theory and algorithm-oriented questions.

  • What are the applications of AI ?
  • Solve the cryptarithmetic problem CSP - SEND + MORE + MONEY
  • Explain hill climbing search algorithm ?
  • Write briefly about adversarial search ?
  • Explain the types of uninformed search algorithm ?
  • Explain the types of Informed search algorithm ?
  • Explain about Heuristic search strategies ?
  • Write about CSP with suitable example ?

Unit II - Probabilistic Solving

This unit introduces probabilistic reasoning and Bayesian learning. Prepare theorem explanations and Bayesian network inference steps for clear exam answers.

  • Explain Bayes' theorem with example ?
  • Explain working of Naive Bayes' Classifier with example ?
  • Explain in detail about exact inference in Bayesian network ?
  • Explain in detail about approximate inference in Bayesian network ?
  • Illustrate causal network with neat diagram ?

Unit III - Supervised Learning

This unit focuses on supervised models and optimization methods. Give special attention to regression, gradient descent, probabilistic models, and decision trees for theory and derivation-style questions.

  • Explain in detail about linear regression model ?
  • Explain the types of linear regression ?
  • Write in detail about least square method ?
  • Write in detail about Bayesian linear regression ?
  • Explain in detail about Gradient Descent ?
  • Write in detail about probabilistic discriminant function ?
  • Write in detail about probabilistic generative model ?
  • Explain in detail about maximum margin classifier ?
  • Write in detail about decision tree ?

Unit IV - Ensemble Techniques and Unsupervised Learning

This unit covers ensemble ideas and unsupervised learning workflows. Students should practice clustering logic, GMM concepts, and EM algorithm steps clearly.

  • Elaborate ensemble techniques in detail ?
  • Explain in detail about instance based learning ?
  • Explain in detail about k-means cluster ?
  • Explain the working of unsupervised learning ?
  • Demonstrate Gaussian mixture model (GMM) ?
  • Discuss on Expectation Maximization (EM) algorithm ?

Unit V - Neural Networks

This unit explains neural network fundamentals and training behavior. Revise perceptron models, activation choices, backpropagation flow, and regularization techniques for strong long-answer preparation.

  • Explain the Basic Components of perceptron ?
  • Demonstrate the multi - layered perceptron model ?
  • Elaborate Activation functions in detail ?
  • Explain Stochastic gradient descent ?
  • What is back propagation ? How backpropagation algorithm works ?
  • Write in detail the vanishing gradient problem ?
  • Disscuss on ReLu ?
  • Demonstrate the strategies for Hyperparameter tuning ?
  • Explain the regularization techniques in machine learning ?
  • Explain about Drop out ?

Exam Preparation Tips

  • Revise one unit daily and keep short notes for each algorithm and formula.
  • Practice search, Bayesian, regression, clustering, and neural network questions regularly.
  • Write algorithm steps clearly with small examples wherever possible.
  • Compare similar methods in a table to improve memory and quick revision.
  • Solve previous year questions to understand repeated patterns and improve speed.
