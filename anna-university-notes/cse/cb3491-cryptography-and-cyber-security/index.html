CB3491 Important Topics (Unit-wise)

Cryptography and Cyber Security is an important subject for understanding how data is protected in modern systems. The topics below are arranged unit wise to help students revise core ideas, practice exam topics, and prepare with confidence.

Unit I – Introduction to Security

  • Classical cryptography techniques (substitution, steganography)
  • Network security basics
  • Types of attacks and security mechanisms
  • Cryptography fundamentals (perfect security, cryptanalysis, information theory)
  • OSI security architecture

Unit II – Symmetric Ciphers

  • Euclid’s algorithm (GCD computation)
  • Symmetric key algorithms (RC4, SDES)
  • Modular arithmetic in cryptography
  • Algebraic structures (groups, rings, fields, finite fields)

Unit III – Asymmetric Cryptography

  • Chinese Remainder Theorem (CRT)
  • RSA algorithm
  • Euler’s Totient function
  • Fermat’s theorem
  • Euler’s theorem

Unit IV – Integrity and Authentication

  • Digital signatures
  • Authentication protocols
  • MAC, HMAC, CMAC
  • SHA hashing algorithm
  • Key management and distribution
  • X.509 certificates

Unit V – Cyber Crimes and Security

  • Types of cyber crimes and lifecycle
  • SQL injection, spyware attacks
  • Wireless, web, and cloud security
  • Phishing, email spoofing, identity theft
  • Cyber law basics and cyber threats
Practice regularly, revise each unit step by step, and focus on concepts as well as problem-solving for better exam results.
