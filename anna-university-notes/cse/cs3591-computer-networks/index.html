CS3591 - Computer Networks Important Questions (Unit-wise)

Computer networking concepts are essential for understanding how devices communicate, share data, and provide internet services in real-world systems. Preparing important questions unit-wise helps students build strong understanding in both theory and protocol-based topics for exams and interviews.

UNIT I

This unit introduces networking foundations, protocol layering, and application-level communication basics. It highlights key concepts such as the OSI model and commonly used network protocols.

  • Network Types – Protocol Layering
  • OSI Model **
  • HTTP, DNS, MIME **

UNIT II

This unit covers reliable and connectionless transport communication in detail. Understanding TCP/IP behavior, UDP operation, and congestion control is important for protocol-based exam questions.

  • TCP, UDP in detail *
  • Congestion Control

UNIT III

This unit explains addressing and packet forwarding in modern networks. Key topics include IP addressing methods, IPv4/IPv6 transition, and packet switching concepts.

  • IP Addressing **
  • DHCP
  • IPv4, IPv6
  • Packet Switching **

UNIT IV

This unit focuses on routing algorithms and path selection in interconnected networks. It is useful for learning how routers make forwarding decisions using dynamic protocols.

  • Distance Vector Routing **
  • OSPF, Path Vector Routing (BGP)
  • DVMRP (Rare)

UNIT V

This unit presents local network technologies and data link communication methods. It covers standards and protocols used in Ethernet, wireless LAN, and media access control.

  • 802.11, 802.3 **
  • Ethernet, WLAN
  • PPP, Media Access Control
  • Data Link Layer Protocol

Exam Preparation Tips

  • Revise one unit per day and prepare short notes for each protocol.
  • Focus on important concepts like OSI model, TCP/IP flow, routing algorithms, and network protocols.
  • Practice protocol comparison questions to improve Part B and Part C answers.
  • Use diagrams for layers, packet flow, and routing to remember concepts quickly.

CS3591 - COMPUTER NETWORKS

L T P C: 3 0 2 4

COURSE OBJECTIVES:

  • To understand the concept of layering in networks.
  • To know the functions of protocols of each layer of TCP/IP protocol suite.
  • To visualize the end-to-end flow of information.
  • To learn the functions of network layer and the various routing protocols
  • To familiarize the functions and protocols of the Transport layer

UNIT I INTRODUCTION AND APPLICATION LAYER 10

Data Communication - Networks - Network Types - Protocol Layering - TCP/IP Protocol suite - OSI Model - Introduction to Sockets - Application Layer protocols: HTTP - FTP - Email protocols (SMTP - POP3 - IMAP - MIME) - DNS - SNMP

UNIT II TRANSPORT LAYER 9

Introduction - Transport-Layer Protocols: UDP - TCP: Connection Management - Flow control - Congestion Control - Congestion avoidance (DECbit, RED) - SCTP - Quality of Service

UNIT III NETWORK LAYER 7

Switching : Packet Switching - Internet protocol - IPV4 - IP Addressing - Subnetting - IPV6, ARP, RARP, ICMP, DHCP

UNIT IV ROUTING 7

Routing and protocols: Unicast routing - Distance Vector Routing - RIP - Link State Routing - OSPF - Path-vector routing - BGP - Multicast Routing: DVMRP - PIM.

UNIT V DATA LINK AND PHYSICAL LAYERS 12

Data Link Layer - Framing - Flow control - Error control - Data-Link Layer Protocols - HDLC - PPP - Media Access Control - Ethernet Basics - CSMA/CD - Virtual LAN - Wireless LAN (802.11) - Physical Layer: Data and Signals - Performance - Transmission media- Switching - Circuit Switching.

45 PERIODS

PRACTICAL EXERCISES: 30 PERIODS

  1. Learn to use commands like tcpdump, netstat, ifconfig, nslookup and traceroute. Capture ping and trace route PDUs using a network protocol analyzer and examine.
  2. Write a HTTP web client program to download a web page using TCP sockets.
  3. Applications using TCP sockets like: a) Echo client and echo server b) Chat
  4. Simulation of DNS using UDP sockets.
  5. Use a tool like Wireshark to capture packets and examine the packets
  6. Write a code simulating ARP /RARP protocols.
  7. Study of Network simulator (NS) and Simulation of Congestion Control Algorithms using NS.
  8. Study of TCP/UDP performance using Simulation tool.
  9. Simulation of Distance Vector/ Link State Routing algorithm.
  10. Simulation of an error correction code (like CRC)

COURSE OUTCOMES:

At the end of this course, the students will be able to:

  • CO 1: Explain the basic layers and its functions in computer networks.
  • CO 2: Understand the basics of how data flows from one node to another.
  • CO 3: Analyze routing algorithms.
  • CO 4: Describe protocols for various functions in the network.
  • CO 5: Analyze the working of various application layer protocols.

TOTAL: 75 PERIODS
